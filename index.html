Сообщение

— информация в виде последовательности сигналов или символов.

Передать или сохранить информацию в "сыром" виде не получится. Для этого ее нужно "превратить" в сообщение. То есть представить в виде последовательности сигналов или символов.

Как измерить количество информации?

Мы знаем, что количество информации измеряется в битах.

Bit — binary digit

1 бит — это количество информации, которое можно передать с помощью одного двоичного знака (0 или 1)

Как измерить количество информации?

1 бит — это количество информации, которое мы получаем при выборе одного из двух возможных вариантов

1 бит — это количество информации, которое уменьшает неопределенность в два раза

Формула Хартли

2i = N

  • i — количество бит
  • N — количество вариантов

Структурирование

— это выделение важных элементов в информационных сообщениях и установление связей между ними.

Структуры данных

  • Множество
  • Список
  • Таблица
  • Граф

Множество

— набор неповторящихся элементов, порядок перечисления которых не важен

Можно задать перечислением элементов:

  • {Вася, Петя, Коля}
  • {1, 33, 45, 20}

Множество

— набор неповторящихся элементов, порядок перечисления которых не важен

А можно описать:

  • множество натуральных чисел
  • n >= 100 and n <= 999

Список

— упорядоченная последовательность элементов (порядок важен).

Таблица

Дерево

— структура данных, служащая для описания иерархии (многоуровневой схемы, где одни элементы подчинены другим).

Дерево

Дерево состоит из узлов (вершин) и ребер (связей между ними).

Корень — самый верхний узел дерева.

Листья — самые нижние узлы, из которых не выходит новых ребер.

Граф

— это набор вершин (узлов) и связей между ними (ребер).

Данные о структуре графа можно записать в виде таблицы смежности.

Петля — ребро, которое начинается и заканчивается в одной и той же вершине.

Степень вершины — количество ребер, с которыми связана эта вершина.

Путь — последовательность ребер, по которым можно перейти из одного узла в другой.

Связный граф — граф, между любыми вершинами которого существует путь.

Цикл — замкнутый путь в графе.

Взвешенный граф

— это граф, с каждым ребром которого связано некоторое число (вес ребра). Оно может означать, например, стоимость проезда или длину дороги. Данные о взвешенном графе хранятся в виде весовой таблицы.

Ориентированный граф

— это граф, в котором ребра имеют направление.

Язык

— это система знаков и правил, используемая для записи и передачи информации.

  • Естественные — языки стран и народов
  • Формальные — искуственно созданные

Язык

— это знаковая система.

Знак — это заменитель какого-то объекта, который вызывает в сознании человека образ этого объекта.

Символ — это знак, о значении которого люди договорились.

Алфавит — набор знаков, используемых в конкретной знаковой системе (языке).
